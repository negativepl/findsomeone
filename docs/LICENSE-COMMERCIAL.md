## Why dual-license?

This model allows:
1. **Free learning & transparency** - Anyone can learn from the code
2. **Sustainable development** - Commercial licenses fund ongoing development
3. **Fair compensation** - Businesses pay for the value they receive
4. **Community contribution** - Non-commercial use remains free forever

This is similar to licensing models used by:
- MongoDB (SSPL)
- Elastic (Elastic License)
- GitLab (MIT + Enterprise)
- Sentry (BSL)
